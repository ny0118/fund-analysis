评分规则说明

EMA交叉信号

快线EMA(12)与慢线EMA(52)的交叉判断趋势方向

EMA12 上穿 EMA52 → +1
EMA12 下穿 EMA52 → -1
无交叉 → 0

MACD信号

基于DIF/DEA的柱状线和零轴判断动量变化

柱状线由负转正 或 DIF上穿零轴 → +1
柱状线由正转负 或 DIF下穿零轴 → -1
无变化 → 0

综合评分

五个周期(日/周/月/季/年)的EMA+MACD信号分别计分

正分和负分分别累加，不抵消
全正 → 偏多 | 全负 → 偏空
有正有负 → 分化 | 全零 → 中性

月线 → 中期趋势
